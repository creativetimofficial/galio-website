# React Native Input Component

Built on top of React Native's TextInput component, this Input offers theme-aware colors, optional icons, password visibility toggle, and helper text.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

![Input](/img/docs/input/input.png)

## Usage

Imports:

```jsx
import { Input } from 'galio-framework';
```

Basic examples:

```jsx
<Input placeholder="regular" />
<Input placeholder="primary" color="primary" />
<Input placeholder="with icon" icon="search" iconFamily="feather" right />
<Input placeholder="password" password viewPass />
```

Advanced examples:

```jsx
<Input placeholder="rounded input" rounded />

<Input
  placeholder="Input with Icon on right"
  right
  icon="heart"
  iconFamily="antdesign"
  iconSize={14}
  iconProps={{ color: 'red' }}
/>

<Input
  placeholder="Input with custom styling and help"
  style={{ borderColor: '#E53935' }}
  help="Bottom help text"
  bottomHelp
  placeholderTextColor="#4F8EC9"
/>
```

## Props

| Prop                     | Type    | Default                        | Description                                                                 |
|--------------------------|---------|--------------------------------|-----------------------------------------------------------------------------|
| **value**                | string  | null                           | The input value (controlled component)                                     |
| **onChangeText**         | function| null                           | Callback when text changes                                                 |
| **type**                 | string  | `'default'`                    | Keyboard type (maps to RN `keyboardType`)                                  |
| **password**             | bool    | false                          | Password input with secure text entry                                      |
| **placeholderTextColor** | string  | `colors.placeholder`           | Placeholder text color                                                      |
| **label**                | string  | null                           | Input label                                                                 |
| **bgColor**              | string  | null                           | Background color (accepts theme keys)                                      |
| **rounded**              | bool    | false                          | Rounded corners                                                             |
| **borderless**           | bool    | false                          | Removes border                                                              |
| **viewPass**             | bool    | false                          | Password visibility toggle                                                   |
| **icon**                 | string  | null                           | Icon name                                                                 |
| **iconProps**            | object  | {}                             | Icon customization object (color, size, family, style)                      |
| **left**                 | bool    | true                           | Icon on the left                                                            |
| **right**                | bool    | false                          | Icon on the right                                                           |
| **error**                | bool    | false                          | Shows error border/color                                                    |
| **help**                 | string  | null                           | Helper text                                                                 |
| **topHelp**              | bool    | true                           | Helper above input                                                          |
| **bottomHelp**           | bool    | false                          | Helper below input                                                          |
| **style**                | object  | null                           | Custom container styles                                                     |
| **textInputStyle**       | object  | null                           | Custom TextInput styles                                                     |
| **labelStyles**          | object  | null                           | Custom label styles                                                         |
| **helpStyles**           | object  | null                           | Custom help styles                                                          |
| **onRef**                | function| null                           | Access to underlying TextInput ref                                          |
| **...TextInput.props**   | -       | -                              | All TextInput props are supported                                           |

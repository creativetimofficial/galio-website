# React Native Button Component

A powerful and highly customizable button built with `Pressable`, designed to look great and perform consistently across iOS, Android, and Web.  
It supports icons, loading states, ripple effects, rounded styles, shadows, and flexible width options.

## Usage

```jsx
import { Button } from 'galio-framework';
```

### Basic Example

```jsx
<Button>Primary</Button>
<Button color="info">Info</Button>
<Button color="success">Success</Button>
<Button color="warning">Warning</Button>
<Button color="error">Error</Button>
<Button color="white">White</Button>
```

### Advanced Examples

```jsx
<Button
  color="primary"
  size="lg"
  shadow="md"
  onPress={() => console.log('Pressed')}
>
  Press Me
</Button>

<Button
  color="success"
  icon="check"
  iconFamily="AntDesign"
  iconSize={18}
  loading={true}
  loadingSize="small"
  loadingColor="white"
>
  Loading
</Button>

<Button
  color="#7B61FF"
  round
  rippleColor="rgba(255,255,255,0.2)"
>
  Custom Color
</Button>

<Button
  color="error"
  icon="close"
  onlyIcon
  size="sm"
  round
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|----------|-------------|
| **children** | `ReactNode` | `undefined` | Button text or content. |
| **color** | `string` | `'primary'` | Background color or predefined theme color (`primary`, `info`, `success`, `warning`, `error`, etc). |
| **disabled** | `boolean` | `false` | Disables button interaction and lowers opacity. |
| **icon** | `string \| boolean` | `undefined` | Icon name (from Expo vector icons). |
| **iconFamily** | `string` | `'AntDesign'` | Icon family (AntDesign, Ionicons, etc). |
| **iconPosition** | `'left' \| 'right'` | `'left'` | Icon placement. |
| **iconSize** | `number` | `16` | Icon size in pixels. |
| **iconColor** | `string` | `theme.colors.white` | Icon color. |
| **onlyIcon** | `boolean` | `false` | Renders only an icon (no text). Useful for round icon buttons. |
| **loading** | `boolean` | `false` | Displays a loading spinner instead of button content. |
| **loadingSize** | `'small' \| 'large'` | `'small'` | Spinner size. |
| **loadingColor** | `string` | `theme.colors.white` | Spinner color. |
| **opacity** | `number` | `0.8` | Button press opacity effect. |
| **rippleColor** | `string` | `'rgba(0,0,0,0.1)'` | Android ripple color. |
| **round** | `boolean` | `false` | Makes the button rounded (circular if `onlyIcon`). |
| **size** | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| 'small' \| 'default' \| 'large'` | `'md'` | Button size (new responsive size system). |
| **fullWidth** | `boolean` | `false` | Makes the button stretch to 100% width of its container. |
| **block** | `boolean` | `false` | Alias for `fullWidth` (kept for backward compatibility). |
| **shadow** | `keyof theme.shadows` | `undefined` | Applies predefined shadow from theme (platform-specific). |
| **style** | `ViewStyle` | `undefined` | Custom button styles. |
| **textStyle** | `TextStyle` | `undefined` | Custom text styles. |
| **textTransform** | `'none' \| 'uppercase' \| 'lowercase' \| 'capitalize'` | `'none'` | Controls text casing. |
| **onPress** | `() => void` | `undefined` | Button press handler. |
| **accessibilityLabel** | `string` | - | Accessibility label for screen readers. |
| **testID** | `string` | - | Test ID for automated testing. |

---

## Notes

- Buttons now use **`Pressable`** instead of `TouchableOpacity` for improved performance and ripple support.
- `block` is deprecated and will be removed in a future release. Use `fullWidth` instead.
- New `shadow` system supports theme-based shadows across platforms.
  - Note: the `shadow` prop expects a key defined in your theme's `shadows` (for example: `sm`, `md`, `lg`). Ensure the key you use exists in your theme.
- The `size` prop supports both **legacy values** (`small`, `large`) and **new ones** (`xs`, `sm`, `md`, `lg`, `xl`).
- The `color` prop accepts **theme color keys** or **custom hex values**.
- When using `onlyIcon`, the button automatically adjusts width and height based on `iconSize`.

## Theming Support

This component is fully integrated with Galio's **Theme Provider**:

```jsx
import { ThemeProvider } from 'galio-framework';

<ThemeProvider
  theme={{
    colors: { primary: '#007BFF', success: '#00D084' },
    sizes: { BASE: 8, BUTTON_HEIGHT: 45, FONT: 14 },
  }}
>
  <Button color="primary">Custom Theme Button</Button>
</ThemeProvider>
```

## Accessibility

The button automatically applies accessibility roles and states:
- `accessibilityRole="button"`
- `accessibilityState={{ disabled }}`
- `accessibilityLabel` defaults to the text content

## Migration from Previous Version

| Old Prop | New Equivalent | Notes |
|-----------|----------------|-------|
| `shadowless` | `shadow={undefined}` | Shadows now use named presets |
| `radius` | `round` | Simplified rounded prop |
| `uppercase`, `lowercase`, `capitalize` | `textTransform` | Unified control |
| `opacity` | same | Works with `Pressable` |
| `block` | `fullWidth` | Alias kept for backward compatibility |

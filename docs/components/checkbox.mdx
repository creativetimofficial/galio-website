# React Native Checkbox Component

Checkboxes allow users to complete tasks that involve making choices such as selecting options, or switching settings on or off. Using the onChange function will enable you to call a function when the state value changes, so basically whenever the user is tapping the Checkbox.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

### Usage

Imports:

```jsx
import { Checkbox } from 'galio-framework';
```

![Checkbox](/img/docs/checkbox/checkbox_simple.png)

Basic examples:

```jsx
<Checkbox color="primary" label="Primary Checkbox" />
<Checkbox color="info" label="Info Checkbox" />
<Checkbox color="error" label="Error Checkbox" />
<Checkbox color="warning" label="Warning Checkbox" />
<Checkbox color="success" label="Success Checkbox" />
```

![Checkbox](/img/docs/checkbox/checkbox_advanced.png)

Advanced examples:

```jsx
<Checkbox color="primary" flexDirection="row-reverse" label="row-reverse checkbox" />
{/* Controlled example (preferred) */}
<Checkbox color="info" checked={true} label="Checked (controlled)" />
{/* Custom icon using iconProps (recommended) */}
<Checkbox color="error" checked={true} label="custom icon" iconProps={{ family: 'FontAwesome', name: 'plane' }} />
<Checkbox color="warning" labelStyle={{ color: '#FF9C09' }} label="labelStyle used here" />
<Checkbox color="success" image="https://images.unsplash.com/photo-1569780655478-ecffea4c165c?ixlib=rb-1.2.1" flexDirection="column-reverse" />
```

Modern usage with hooks (controlled):

```jsx
const [isChecked, setIsChecked] = useState(false);

<Checkbox
  color="warning"
  checked={isChecked}
  onChange={setIsChecked}
  label="onChange is here"
 />
```

### Props

| Prop              | Type    | Default       | Description                                                                                           |
| ----------------- | ------- | ------------- | ----------------------------------------------------------------------------------------------------- |
| **label**         | string  | null          | Checkbox label text                                                                                   |
| **color**         | string  | 'primary'     | Checkbox color (theme key or hex string)                                                              |
| **checked**       | bool    | false         | Controlled checkbox value (use this for controlled components)                                        |
| **onChange**      | function| null          | Callback when checkbox state changes; receives the new checked state (boolean)                        |
| **initialValue**  | bool    | false         | (Deprecated) Initial uncontrolled checkbox value — prefer using controlled `checked` or `iconProps`   |
| **disabled**      | bool    | false         | Disables the checkbox                                                                                 |
| **flexDirection** | string  | 'row'         | Layout direction: 'row', 'row-reverse', 'column', 'column-reverse'                                    |
| **iconProps**     | object  | null          | Icon customization object — fields: `name`, `family`, `color`, `size` (recommended over individual icon props) |
| **image**         | string  | null          | Custom image URL                                                                                      |
| **imageStyle**    | object  | null          | Custom image styles                                                                                    |
| **checkboxStyle** | object  | null          | Custom checkbox (box) styles                                                                          |
| **labelStyle**    | object  | null          | Custom label styles                                                                                   |
| **labelColor**    | string  | null          | Color for the label (theme key or hex string)                                                        |
| **style**         | object  | null          | Custom container styles (wrapper for checkbox + label)                                               |
| **accessibilityLabel** | string | -        | Accessibility label for screen readers                                                                |
| **accessibilityHint**  | string | -        | Accessibility hint for assistive technologies                                                         |

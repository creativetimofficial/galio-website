# React Native Deck Swiper Component

Swipe left, swipe right or don't swipe at all, this component is built to accept any type of component and make it swipeable. Galio's community plans on creating more props for easier access and manipulation of the swipe animation.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

### Usage

Imports:

```jsx
import { DeckSwiper, Block } from 'galio-framework';
```

Simple example:

```jsx
<Block>
  <DeckSwiper components={elements} />
</Block>
```

The elements array looks like following:

```jsx
const elements = [
  <Block style={{ backgroundColor: '#B23AFC', height: 250, width: 250 }}>
    <Text>You wanna see a cool component?</Text>
    <Text>Galio has this cool Deck Swiper</Text>
  </Block>,
  <Block style={{ backgroundColor: '#FE2472', height: 250, width: 250 }}>
    <Text>What did you expect?</Text>
    <Text>This React Native component works perfectly</Text>
  </Block>,
  <Block style={{ backgroundColor: '#FF9C09', height: 250, width: 250 }}>
    <Text>Maybe you want to build the next Tinder</Text>
  </Block>,
  <Block style={{ backgroundColor: '#45DF31', height: 250, width: 250 }}>
    <Text>or maybe you just want a nice deck swiper component</Text>
    <Text>Galio has everything sorted out for you</Text>
  </Block>
];
```

Advanced examples:

```jsx

<DeckSwiper
  components={elements}
  onSwipeLeft={(index) => console.log('Swiped left on:', index)}
  onSwipeRight={(index) => console.log('Swiped right on:', index)}
  style={{ height: 300 }}
  cardContainerStyle={{ borderRadius: 16 }}
  cardShadow="md"
  cardWidth={300}
  cardBackgroundColor="#fff"
  nextCardBackgroundColor="#f7f7f7"
  nextCardShadow="sm"
  focusedElementStyle={{ elevation: 6 }}
  nextElementStyle={{ transform: [{ scale: 0.95 }] }}
  borderRadius={16}
  showNextCard={true}
/>
```

### Props

| Prop                   | Type                       | Default                         | Description                                                                 |
|------------------------|----------------------------|---------------------------------|-----------------------------------------------------------------------------|
| **components**         | `React.ReactNode[]`       | required                        | Array of components to render in the deck                                   |
| **onSwipeLeft**        | `(index: number) => void` | `() => {}`                      | Callback when swiping left (receives the card index)                       |
| **onSwipeRight**       | `(index: number) => void` | `() => {}`                      | Callback when swiping right (receives the card index)                      |
| **style**              | `object` / `StyleProp<ViewStyle>` | `null`                 | Container styles                                                            |
| **cardContainerStyle** | `object` / `StyleProp<ViewStyle>` | `null`                 | Styles applied to the focused card container                                |
| **cardWidth**          | `number`                   | `SCREEN_WIDTH * 0.7`            | Card width in pixels                                                        |
| **cardShadow**         | `string` / `object`   | `'md'`                          | Theme shadow key (e.g. `sm`, `md`, `lg`) or a shadow style object for the focused card |
| **cardBackgroundColor**| `string`                   | `theme.surface` / `'#fff'`      | Background color of the focused card                                        |
| **nextCardBackgroundColor** | `string`            | `theme.background` / `'#f7f7f7'`| Background color for the next card preview                                  |
| **nextCardShadow**     | `string` / `object`   | `'sm'`                          | Theme shadow key or style object for the next card                          |
| **swipeThreshold**     | `number`                   | `110`                           | Distance required to trigger a swipe                                        |
| **focusedElementStyle**| `object` / `StyleProp<ViewStyle>` | `{}`                  | Additional styles for the focused (top) card                               |
| **nextElementStyle**   | `object` / `StyleProp<ViewStyle>` | `{}`                  | Styles for the next card preview                                            |
| **borderRadius**       | `number`                   | `theme.sizes.CARD_BORDER_RADIUS`| Card border radius                                                          |
| **showNextCard**       | `boolean`                  | `true`                          | Whether to render the next card peek                                        |
| **...View.props**      | -                          | -                               | All React Native `View` props are supported                                 |

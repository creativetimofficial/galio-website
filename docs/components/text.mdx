# React Native Text Component

It's never been easier to write and style text. We're using a helper function to normalize the text so that it could look good on any kind of device.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

![Text](/img/docs/text/text.png)

### Usage

Imports:

```jsx
import { Text } from 'galio-framework';
```

Basic examples:

```jsx
<Text h1>Heading 1</Text>
<Text h2>Heading 2</Text>
<Text h3>Heading 3</Text>
<Text p>Paragraph</Text>
<Text muted>This is a muted paragraph.</Text>
```

Advanced examples:

```jsx
<Text 
  h1 
  color="primary" 
  bold 
  style={{ textAlign: 'center' }}
>
  Custom Styled Heading
</Text>

<Text 
  p 
  muted 
  italic 
  size={18}
  style={{ lineHeight: 24 }}
>
  Custom paragraph with specific styling
</Text>
```

### Props

| Prop      | Type    | Default | Description                                   |
| --------- | ------- | ------- | --------------------------------------------- |
| **h1**    | bool    | false   | Heading 1 style (large)
| **h2**    | bool    | false   | Heading 2 style
| **h3**    | bool    | false   | Heading 3 style
| **h4**    | bool    | false   | Heading 4 style
| **h5**    | bool    | false   | Heading 5 style
| **p**     | bool    | false   | Paragraph style
| **size**  | number  | undefined | Explicit font size (overrides semantic sizes)
| **color** | string  | undefined | Color or theme key
| **muted** | bool    | false   | Applies muted text color
| **bold**  | bool    | false   | Bold text
| **italic**| bool    | false   | Italic text
| **center**| bool    | false   | Center text
| **style** | object  | null    | Custom styles

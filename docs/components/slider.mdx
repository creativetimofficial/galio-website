# React Native Slider Component

We all need a slider in our application. This is a JavaScript component built with the PanResponder and Animated APIs.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

![Slider](/img/docs/slider/slider.png)

### Usage

Imports:

```jsx
import { Slider } from 'galio-framework';
```

Simple example:

```jsx
<Block flex>
  <Slider
    maximumValue={30}
    value={10}
    onSlidingComplete={() => this.registerTheValue()}
  />
</Block>
```

Modern usage with hooks:

```jsx
const [sliderValue, setSliderValue] = useState(0.5);

<Slider
  value={sliderValue}
  onValueChange={setSliderValue}
  minimumValue={0}
  maximumValue={1}
  step={0.01}
  activeColor="primary"
/>
```

### Props

| Prop             | Type      | Default                        | Description                                                                                      |
| ---------------- | --------- | ------------------------------ | ------------------------------------------------------------------------------------------------ |
| **value**        | number    | `0`                            | The current value of the slider (controlled)                                                    |
| **onValueChange**| function  | null                           | Callback when slider value changes                                                               |
| **minimumValue** | number    | `0`                            | Minimum slider value                                                                             |
| **maximumValue** | number    | `1`                            | Maximum slider value                                                                             |
| **step**         | number    | `0.01`                         | Step increment value                                                                             |
| **disabled**     | bool      | `false`                        | Disables the slider                                                                              |
| **trackStyle**   | object    | `{}`                           | Custom track styles                                                                              |
| **thumbStyle**   | object    | `{}`                           | Custom thumb styles                                                                              |
| **containerStyle** | object  | `{}`                           | Container style                                                                                  |
| **activeColor**  | string    | `primary`                      | Active track color (theme key or hex)                                                            |
| **accessibilityLabel** | string | null                        | Accessibility label                                                                               |

# React Native Toast Notification Component

Toast notifications appear when the user finishes an action and you want to show him a piece of information without redirecting him to another screen.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

### Usage

Imports:

```jsx
import { Toast } from 'galio-framework';
```

Simple example:

```jsx
const [isShow, setShow] = useState(false);

<Button onPress={() => setShow(!isShow)}>Show toast</Button>
<Toast isShow={isShow} positionIndicator="top">This is a top positioned toast</Toast>
```

Advanced example:

```jsx
<Toast
  isShow={isShow}
  positionIndicator="top"
  color="success"
  round
>
  Operation completed successfully
</Toast>

<Toast isShow={isShow} positionIndicator="center" color="error">Something went wrong</Toast>
```

### Props

| Prop                | Type       | Default    | Description                                                                                         |
| ------------------- | ---------- | ---------- | --------------------------------------------------------------------------------------------------- |
| **isShow**          | bool       | false      | Controls toast visibility                                                                           |
| **positionIndicator** | string  | 'top'      | Toast position: 'top', 'center', 'bottom'                                                          |
| **positionOffset**  | number     | 120        | Offset used to calculate top/bottom position                                                      |
| **fadeInDuration**  | number     | 300        | Fade-in animation duration (ms)                                                                    |
| **fadeOutDuration** | number     | 300        | Fade-out animation duration (ms)                                                                   |
| **color**           | string     | 'primary'  | Toast color theme (theme key or color string)                                                      |
| **round**           | bool       | false      | Use rounded corners                                                                                 |
| **style**           | object     | null       | Custom styles                                                                                       |
| **textStyle**       | object     | null       | Custom text style                                                                                   |

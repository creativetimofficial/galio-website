# React Native Card Component

Card is a flexible and extensible content container for React Native. It supports headers, images, avatars, captions, and interactive content areas. You can use it to display user profiles, posts, product information, or any other visual block.

<CarbonAds serve="CE7IPK3L" placement="galioio" />

### Import

```jsx
import { Card } from 'galio-framework';
```

### Basic Example

```jsx
<Card
  flex
  borderless
  title="Christopher Moon"
  caption="139 minutes ago"
  location="Los Angeles, CA"
  avatar="http://i.pravatar.cc/100?id=skater"
  image="https://images.unsplash.com/photo-1497802176320-541c8e8de98d?&w=1600&h=900&fit=crop&crop=entropy&q=300"
/>
```

### Advanced Example

```jsx
<Card
  flex
  card
  shadow
  borderless
  title="Custom Card"
  caption="With custom styling"
  titleColor="primary"
  captionColor="muted"
  location="New York, USA"
  rightSideComponent={<Button size="small" color="primary">Follow</Button>}
  footerStyle={{ paddingTop: 10 }}
  onPress={() => console.log("Card pressed!")}
/>
```

### Full Background Image Card

```jsx
<Card
  fullBackgroundImage
  image="https://images.unsplash.com/photo-1557682224-5b8590cd9ec5"
  title="Nature Escape"
  caption="A peaceful getaway among the mountains."
  authorImageSrc="https://i.pravatar.cc/100?id=nature"
  authorTitle="Jane Doe"
  authorSubTitle="Travel Blogger"
  onPress={() => console.log("Opened card")}
/>
```

### Props

| Prop | Type | Default | Description |
|------|------|----------|-------------|
| **avatar** | string | - | URL for the avatar image. |
| **title** | string | - | Card title text. |
| **titleColor** | string | - | Color of the title text. |
| **caption** | string | - | Short subtitle or secondary text. |
| **captionColor** | string | - | Color of the caption text. |
| **location** | string | - | Location text shown with an icon. |
| **locationColor** | string | - | Custom color for the location text/icon. |
| **image** | string | - | URL for the main image. |
| **imageBlockStyle** | object | - | Custom styles for the image container. |
| **imageStyle** | object | - | Custom styles for the image. |
| **footerStyle** | object | - | Custom styles for the footer area. |
| **rightSideComponent** | ReactNode | - | Component rendered on the right side of the header (e.g. button). |
| **borderless** | bool | false | Removes border around the card. |
| **card** | bool | true | Enables card-style background. |
| **shadow** | bool | true | Adds subtle shadow. |
| **neutral** | bool | false | Transparent, borderless card variant. |
| **fullBackgroundImage** | bool | false | Uses the image as the entire background. |
| **authorImageSrc** | string | - | Alternate avatar image source (used for author cards). |
| **authorTitle** | string | - | Alternate title (used for author cards). |
| **authorSubTitle** | string | - | Alternate caption (used for author cards). |
| **onPress** | function | - | Function called when the card is pressed. |
| **flex** | bool | false | Makes the card expand to fill available space. |
| **shadowColor** | string | - | Custom shadow color. |
| **style** | object | - | Custom styles for the card container. |
| **children** | ReactNode | - | Custom JSX content inside the card. |
| **...View.props** | - | - | All native View props are supported. |

### Notes

- The `Card` component is designed to be **theme-aware**, adapting colors and spacing from your theme.
- For clickable cards, wrap content with `onPress` instead of a separate `TouchableOpacity`.
- Works seamlessly with the `Block`, `Button`, and `Text` components from Galio.

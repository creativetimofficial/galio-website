## React Native Accordion Component

This accordion component made by Galio is easily customizable. Built with Flatlist, this expandable list can be used to only show relevant information and save space in your screen.

### Usage

Imports:

```jsx
import { Accordion } from 'galio-framework';
```

Simple example:

```jsx
const data = [
  { title: 'First', content: 'First content' },
  { title: 'Second', content: 'Second content' },
];

<Accordion dataArray={data} opened={0} />
```

The data array looks like:

```jsx
const data = [
  { title: "First Chapter", content: "Lorem ipsum dolor sit amet", 
    icon: {
      name: 'keyboard-arrow-up',
      family: 'material',
      size: 16,
    } 
 },
  { title: "2nd Chapter", content: "Lorem ipsum dolor sit amet" },
  { title: "3rd Chapter", content: "Lorem ipsum dolor sit amet" }
];
```

Advanced examples:

```jsx
// Simple: open first item
<Accordion
  dataArray={data}
  opened={0}
  onAccordionOpen={(item, index) => console.log('open', index)}
  onAccordionClose={(item, index) => console.log('close', index)}
/>

// Custom icons and styles
<Accordion
  dataArray={data}
  icon={{ name: 'chevron-down', family: 'feather' }}
  expandedIcon={{ name: 'chevron-up', family: 'feather' }}
  headerStyle={{ padding: 12 }}
  contentStyle={{ padding: 16 }}
  shadow="sm"
/>
```

### Props

| Prop            | Type     | Default | Description                                                                                                     |
|-----------------|----------|---------|-----------------------------------------------------------------------------------------------------------------|
| **dataArray**   | array    | null    | Array of items: \{ title, content, icon? \}                                                                       |
| **opened**      | number   | undefined | Index of item initially opened                                                                                 |
| **icon**        | object   | undefined | Default icon for collapsed items (object with name/family/size/color)                                          |
| **expandedIcon**| object   | undefined | Icon shown for expanded items                                                                                    |
| **headerStyle** | object   | undefined | Style for header row                                                                                             |
| **contentStyle**| object   | undefined | Style for content area                                                                                           |
| **onAccordionOpen** | function | undefined | Callback (item, index) when opened                                                                               |
| **onAccordionClose**| function | undefined | Callback (item, index) when closed                                                                               |
| **listStyle**   | object   | undefined | Style for the wrapping list container                                                                            |
| **style**       | object   | undefined | Container styles                                                                                                 |
| **titleStyle**  | object   | undefined | Title text style                                                                                                 |
| **shadow**      | string   | 'md'     | Semantic shadow level (none | xs | sm | md | lg | xl)                                                              |

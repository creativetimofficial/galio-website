## Cómo instalar

Simplemente agrégalo como una dependencia npm/yarn:

```bash
npm install galio-framework o yarn add galio-framework
```

Y solo importa el componente que quieres en tu aplicación:

```jsx
import { Text } from 'galio-framework'
```

¿Esto parece realmente fácil, verdad? Solo usa tu componente como normalmente lo harías con cualquier otro componente.

```jsx
render() {
  return (
    <View>
      <Text p muted>Hola, soy un componente Galio</Text>
    </View>
  );
}
```
